### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mare-mrgazo/sahomat.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
### Usage

1. Run server.js
   ```sh
   node server.js
   ```
